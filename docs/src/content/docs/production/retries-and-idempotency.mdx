---
title: Retries and idempotency
description: How the SDK retries transient failures and how to make your render calls safe to retry.
---

The Poli Page Go SDK retries transient failures automatically. Render and thumbnail operations carry an auto-generated UUID v4 idempotency key, so retries do not produce duplicate stored documents on the server.

## Default retry policy

- `network_error`, `timeout`, 5xx responses, and 429 (`QUOTA_EXCEEDED`) are retried.
- Up to `option.WithMaxRetries(n)` attempts (default 2 — three total attempts).
- Exponential backoff starting at `option.WithRetryDelay(d)` (default `500ms`), doubling each attempt with jitter.
- Honors the `Retry-After` response header when present.
- `aborted` (context-cancelled) is never retried.

## Disabling retries

Pass `option.WithMaxRetries(0)` to disable retries entirely.

```go
client := polipage.NewClient(
	option.WithAPIKey(os.Getenv("POLI_PAGE_API_KEY")),
	option.WithMaxRetries(0),
)
```

## Idempotency

Every render and thumbnail request includes a UUID v4 `Idempotency-Key` header generated by the SDK. The API treats requests with the same key as the same operation, so retries from inside the SDK never create duplicate stored documents.

If you orchestrate your own retries on top of the SDK (not recommended), supply your own deterministic key via `option.WithIdempotencyKey` as the last argument:

```go
pdf, err := client.Render.PDF(ctx, polipage.ProjectModeInput{
	Project:  "billing",
	Template: "invoice",
	Data:     map[string]any{"invoiceNumber": "INV-001"},
}, option.WithIdempotencyKey("invoice-INV-001-v1"))
```

Same input + same key → same stored document. Different key with same data → new document.

## Example

```go
package main

import (
	"context"
	"os"
	"time"

	"github.com/poli-page/sdk-go"
	"github.com/poli-page/sdk-go/option"
)

func main() {
	client := polipage.NewClient(
		option.WithAPIKey(os.Getenv("POLI_PAGE_API_KEY")),
		option.WithMaxRetries(4),
		option.WithRetryDelay(250*time.Millisecond),
	)

	if _, err := client.Render.PDF(context.Background(), polipage.ProjectModeInput{
		Project:  "billing",
		Template: "invoice",
		Data:     map[string]any{"invoiceNumber": "INV-001"},
	}); err != nil {
		panic(err)
	}
}
```
