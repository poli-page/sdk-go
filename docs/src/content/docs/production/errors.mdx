Every failure returned by the SDK is a `*polipage.Error`. Use `errors.As` to extract typed fields and `errors.Is` against the package-level sentinels (`ErrNotFound`, `ErrUnauthorized`, …) for quick branching.

## The error type

```go
var perr *polipage.Error
if errors.As(err, &perr) {
	fmt.Println(perr.Code, perr.StatusCode, perr.RequestID, perr.Message)
}
```

`*Error` implements `Unwrap`, so `errors.Unwrap(err)` gives you the underlying I/O / context cause when present.

## SDK-internal codes

These originate inside the SDK before any HTTP call (or in response to a network-layer failure). They are lowercase.

## Example

```go
package main

import (
	"context"
	"errors"
	"fmt"
	"os"

	"github.com/poli-page/sdk-go"
	"github.com/poli-page/sdk-go/option"
)

func main() {
	client := polipage.NewClient(option.WithAPIKey(os.Getenv("POLI_PAGE_API_KEY")))

	_, err := client.Render.PDF(context.Background(), polipage.ProjectModeInput{
		Project:  "billing",
		Template: "invoice",
		Data:     map[string]any{"invoiceNumber": "INV-001"},
	})
	if errors.Is(err, polipage.ErrRateLimit) {
		// Custom strategy beyond the SDK's built-in retries.
		return
	}
	if err != nil {
		var perr *polipage.Error
		if errors.As(err, &perr) {
			fmt.Println("request id:", perr.RequestID)
		}
		panic(err)
	}
}
```
