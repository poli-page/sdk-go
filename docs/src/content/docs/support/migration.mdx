This page tracks breaking changes between major versions of `github.com/poli-page/sdk-go`. For the full per-release diff, see the [changelog](../changelog/).

## v1.0

The first stable release. The full v1.0 surface and renames from the pre-1.0 line are documented in [`MIGRATION.md` in the repo](https://github.com/poli-page/sdk-go/blob/main/MIGRATION.md).

Short version: every render call goes through the `Render` namespace (`Render.PDF`, `Render.Preview`, `Render.Document`, `Render.PDFStream`), the PDF-producing methods accept only `ProjectModeInput`, and inline mode is supported only on `Render.Preview`.
