---
title: Render modes
description: Project mode renders a stored template; inline mode renders raw HTML and is supported only by Render.Preview.
---

The Go SDK exposes two ways to describe what to render: **project mode** (a stored template at `project/template`) and **inline mode** (raw HTML you pass directly).

Project mode is required by all the PDF-producing methods (`Render.PDF`, `Render.PDFStream`, `Render.Document`, and the `RenderToFile` helper) — their signatures only accept `ProjectModeInput`. Inline mode is supported only by `Render.Preview`, which returns HTML rather than PDF bytes and is useful for debugging template data without engaging the PDF pipeline.

## Project mode

You reference a template by its `Project` and `Template` slugs, plus a `Data` map that fills the template's placeholders. Optionally pin a version with `polipage.Opt`.

```go
pdf, err := client.Render.PDF(ctx, polipage.ProjectModeInput{
	Project:  "billing",
	Template: "invoice",
	Data:     map[string]any{"invoiceNumber": "INV-001", "total": 1280},
	Version:  polipage.Opt("1.0.0"),
})
```

Project mode keeps templates centralized — anyone with dashboard access can edit them without touching code.

## Inline mode (preview only)

For one-off layout experiments, pass HTML directly via `InlineModeInput`. The `Template` field accepts raw HTML when there's no `Project`:

```go
preview, err := client.Render.Preview(ctx, polipage.InlineModeInput{
	Template: "<h1>Hello {{ name }}</h1>",
	Data:     map[string]any{"name": "World"},
})
```

`Render.Preview` is the only method that accepts `InlineModeInput`. To turn inline HTML into a PDF, save it as a template in the dashboard first, then use project mode.

## Example

```go
package main

import (
	"context"
	"os"

	"github.com/poli-page/sdk-go"
	"github.com/poli-page/sdk-go/option"
)

func main() {
	ctx := context.Background()
	client := polipage.NewClient(option.WithAPIKey(os.Getenv("POLI_PAGE_API_KEY")))

	// Project mode — PDF.
	if _, err := client.Render.PDF(ctx, polipage.ProjectModeInput{
		Project:  "billing",
		Template: "invoice",
		Data:     map[string]any{"invoiceNumber": "INV-001", "total": 1280},
	}); err != nil {
		panic(err)
	}

	// Inline mode — preview HTML only.
	if _, err := client.Render.Preview(ctx, polipage.InlineModeInput{
		Template: "<h1>Hello {{ name }}</h1>",
		Data:     map[string]any{"name": "World"},
	}); err != nil {
		panic(err)
	}
}
```
