# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities to **security@poli.page**.

Do not file public GitHub issues for security concerns.
We aim to respond within 48 hours.

## Supported Versions

Only the latest minor version of `github.com/poli-page/sdk-go` receives
security updates. The Go module proxy serves immutable versions — old
releases remain installable but are not patched in place.
